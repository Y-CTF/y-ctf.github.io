## TryHackMe

[TryHackMe](https://tryhackme.com) is a platform with interactive cyber security training.

- **[Easy CTF](https://tryhackme.com/room/easyctf)** - A beginner-friendly CTF challenge
- **[RRootMe](https://tryhackme.com/room/rrootme)** - Classic privilege escalation challenges
- **[OhSINT](https://tryhackme.com/room/ohsint)** - Open Source Intelligence reconnaissance
- **[Sakura](https://tryhackme.com/room/sakura)** - Real-world OSINT techniques (pair with OhSINT)
- **[Pickle Rick](https://tryhackme.com/room/picklerick)** - A fun exploit challenge
- **[Takeover](https://tryhackme.com/room/takeover)** - Subdomain enumeration and takeover

## Root Me

[Root Me](https://root-me.org) offers a variety of challenges from beginner to advanced levels.

Start with the [**Challenges**](https://www.root-me.org/fr/Challenges/) section and explore:
- Web exploitation challenges
- Linux privilege escalation
- Cryptography exercises
- Network analysis
